# Contributing Guidelines

## Branch Naming
- feature/<issue-id>-short-description
- bugfix/<issue-id>-short-description
- chore/<short-description>

## Pull Request Rules
- Link related issue
- Ensure CI passes
- Add tests for new features
- No direct push to main

## Commit Convention
Use:
- feat:
- fix:
- chore:
- refactor:
- docs:
- test:
